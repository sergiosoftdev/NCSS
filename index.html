Box Types

Advice Type Box

Headers

Headers are optimized for a minimalist view. There's only one header that can be declared and is is Section Headers. The way to declare a section header is using an <h1> and calling the class named "section-header".

Example

<h1 class="section-header">Section Header</h1>

Text Areas

Text Areas in NCSS are always 100% the width of the father element. You should use the grid or box system if you want an element layed in the same row as the text area.

Light Text Area

Light Text Areas class are declared as 'light-textarea'.

Dark Text Area

Dark Text Areas class are declared as 'dark-textarea'.
